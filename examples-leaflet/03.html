03 Capas vectoriales (csv)

Carga datos (csv) desde una variable local como una capa geojson

Importar Leaflet plugin
Una extensión (plugin) se importa como cualquier fichero JavaScript mediante script en la página HTML, justo después de importar la librería Leaflet.
Declarar variable local con datos csv
El valor de la variable csvData es una lista de paradas de autobus. Cada una de las paradas está separada por \n, y cada uno de los attributos de una parada esta separado por comas
Cargar la varaible local como una capa geojson
Se llama a L.geoCsv(). El primer parámetro es el contenido de la variable local (los datos), y el segundo parámetro define un objeto de configuración. Una vez la capa está cargada, se añade al mapa con map.addLayer().
