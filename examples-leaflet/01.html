01 Mapas base

Carga y muestra una capa base en un mapa

Map initialization
Declare the starting parameters for the map with L.Map(). First parameter is an div element id (e.g. map). Second parameter is optional and corresponds to an object that contains map state options.
Center a map
Set the center of a map by using map.setView() by passing an object LatLon as input parameter. Finding the pair of coordinates, latlon, of the centre of a map seems an obstacle. One can get them from diverse services on the Web such like geojson.io, a native geojson editor to encode geometries (point, lines, polygons) in geojson format. Note that in geojson the longitude comes first and then latitude while Leaflet follows the order latitude and longitude of the centre point.
Add a base map layer
Declare the source of the map tiles with L.tileLayer(). First parameter is a URL template. Second parameter is the TileLayer options that provide a range of different options when loading or displaying the tiles such as attribution and maxZoom. Attribution is the placing of appropriate reference to the source of the tiles.
