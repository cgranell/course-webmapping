01 Base maps

Things to remember:

Map initialization
Declare the starting parameters for the map with L.Map(). First parameter is an id element (e.g. map). Second parameter is optional and corresponds to an object that contains map options.
Center a map
Set the center of a map by using map.setView() by passing an object LatLon as input parameter.
Add a base map layer
Declare the source of the map tiles with L.tileLayer()
.

Key methods:

  • L.Map()
  • map.setView()
  • L.tileLayer()
