02 Mapas base

Control de capas para la selección de mapas base

Declare various base map layer
Declare the source of the map tiles with L.tileLayer(), one call for each base map to add.
Set active a base layer
The default base map layer is set by calling addMap(). Otherwise, the map is being displayed without any base map until we select one from the Layer control.
Add a layer control to the map
A layer control manages basemap layers and overlayers. It is created by calling L.control.layers() with an object Layer Config as input parameter. It contains a list of base map layers and overlayers. Once the control layer is initialized, it is added to the map with map.addControl().

Key methods

  • L.control.layers()
  • map.addControl()
  • L.tileLayer()
